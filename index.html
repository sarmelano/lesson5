Галерея
  1. git init = создать локальный репозиторий/подключить папку к git.
  2. git status = проверить состояние.
  3. git checkout = откат.
  4. git add = добавить в "stage" зону для последущей фиксации.
  5. git commit -m "комментарий" = произвести фиксацию из "stage" зоны.
  6. git commit -am "комментарий" = произвести фиксацию включая файлы вне "stage" зоны.
  7. git diff = отобразить изминения.
  8. git log = показать всю историю изминений в репозитории "покоммитно" с номерами коммитов.
  9. git log -n кол-во = показать количество последних изминений в репозитории с номерами коммитов.
  10. git revert ...номер коммита... = откатить данное изминение/коммит.
